The `matrix_divided` function
=============================

The `matrix_divided` function takes two arguments, `matrix` and `div`, and returns a new matrix where each element is divided by `div`. The `matrix` argument should be a list of lists, where each inner list represents a row of the matrix, and each row should have the same number of elements. The `div` argument should be a number (integer or float). 

Function signature:
-------------------
```python
def matrix_divided(matrix, div):
    """
    Returns a new matrix where each element is divided by `div`.

    :param matrix: a list of lists, where each inner list represents a row of the matrix
    :param div: a number (integer or float)
    :return: a new matrix where each element is divided by `div`
    """
```

Raises
------
This function raises the following exceptions:

    ZeroDivisionError: If `div` is zero
    TypeError: If the matrix contains any non-numeric values, or if the rows of the matrix have different lengths

Usage
-----
To use the `matrix_divided` function, first import it from the `matrix_divided` module:

```python
from matrix_divided import matrix_divided
```

Then call the `matrix_divided` function with a list of lists and a number:

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

div = 2

result = matrix_divided(matrix, div)

print(result)
```

This will output:

```
[[0.5, 1.0, 1.5], [2.0, 2.5, 3.0], [3.5, 4.0, 4.5]]
```

result = matrix_divided(matrix, div)
